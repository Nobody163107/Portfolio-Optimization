# Portfolio-Optimization

Portfolio Optimization is an optimization algorithm to invest in the right stocks so as to increase the returns and minimize the risk 

### Decision Variables:
It contains the decision variables x1, x2..., xn which are the investments to be done in the n assets 1, 2, ..., n. w1, w2..., wn are the weights to be used to assign the fraction of our budget to each investment.

### Constraints:
There would be two constraints::
1. Expected Return: This is the minimum value of return we expect after investing in the stock, it lies between the maximum return and the minimum return of the stock. 
                <!-- Here goes the expression in latex  -->

2. Weights: This is the maximum amount of weight we can assign to each each asset, this reduces the chance of investing completely in one stock expecting high return.
